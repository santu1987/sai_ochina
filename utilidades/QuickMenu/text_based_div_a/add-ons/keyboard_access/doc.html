Keybaord Access Add-On
Keyboard access allows for the complete navigation of your menu using the tab key, arrow keys, and enter key. The arrow keys move the focus between items and open sub menus which exist in the direction of movement. The enter key activates links or opens child sub menus for an item. The tab key and shift+tab keys move the focus back and forth between the items.
Applying the Add-On
The add-on is applied by including the core add-on code and the script reference to the 'qm_keybaord_access.js" file within your HTML. The following code must be added directly after your documents call to the 'qm.js' file...
<!--%%%%%%%%%%%% QuickMenu JavaScript %%%%%%%%%%%*-->

<!-- Core Menu Code -->
<script type="text/JavaScript" src="qm.js"></script>

<!-- Add-On Core Code (Remove when not using any add-on's) -->
<style type="text/css">.qmfv{visibility:visible !important;}</style><script type="text/JavaScript">var qmad = new Object();qmad.bvis="";qmad.bhide="";</script>

<!-- Add-On Settings -->
<script type="text/JavaScript">

/******* Menu 0 Add-On Settings *******/
var a = qmad.qm0 = new Object();

    // Keyboard Access Add On
    a.keyboard_access_active = true;

</script>

<!-- Optional Add-On's (Must appear after core menu code) -->
<script type="text/JavaScript" src="qm_keyboard_access.js"></script>
 
*Note: It is not necessary to duplicate the core add-on code if you already have it included in your document.
Optional Hover Styles
Hover styles may be applied by adding a class name rule to your menus CSS styles. The hover styles will be visible when navigating the structure. Add the portions in red to your menus "main hover" and "sub hover" CSS styles...
/*"""""""" (MAIN) Hover State """"""""*/
#qm0 a:hover, #qm0 .qmkeyboardaccess
{...


  /*"""""""" [SUB] Hover State """"""""*/
#qm0 div a:hover, #qm0 div .qmkeyboardaccess
{...

*Note: Remember to include the comma which seperates the rules, without it your menu will not function correctly.
