QuickMenu Image Based Items

View the source HTML for this page to see where to apply this add on to your menu, or use this template as your starting point.

Image based items are created by adding a standard image tag with a special class name to any menu item. Each image tag must include the class qm-is...

<img class="qm-is qm-ih qm-ia" src="images/cat50.jpg" width="110" height="82">

Optionally you may include qm-ih which activates a hover image, and qm-ia which activates an active image. The hover and active images must be in the same folder as your static image and have '_hover' or '_active' appended to the file name (e.g. main_image1_hover.gif).

You must also include the item image add-on, see the source of this document for a first hand look.
